# ISA 완전정복 체크리스트

> 🐻 곰곰 아저씨의 무료자료 #1 — 가입 전 확인부터 3년 유지 전략까지, 이 페이지 하나면 끝.

## STEP 1 · 가입 전 확인

- [ ] 가입 자격 — 만 19세 이상 (근로소득 있으면 15세부터)
- [ ] 유형 선택 — 일반형 / 서민형(총급여 5천만원 이하, 비과세 한도↑)
- [ ] 금융소득종합과세 대상이면 가입 불가 — 해당 여부 확인
- [ ] 중개형 vs 신탁형 — 주식·ETF 직접 투자하려면 '중개형'

## STEP 2 · 계좌 개설 (10분)

- [ ] 증권사 앱에서 ISA(중개형) 검색 → 비대면 개설
- [ ] 서민형 대상자는 소득확인증명서 제출 (홈택스 발급)
- [ ] 수수료·이벤트 비교 — 증권사별 ISA 이벤트 확인

> 💡 곰곰 팁: ISA는 전 금융기관 통틀어 1인 1계좌예요. 옮기고 싶으면 '이전' 제도를 이용하세요.

## STEP 3 · 운용

- [ ] 자동이체 설정 — 월급날 다음 날로 (연 납입한도 확인)
- [ ] 처음엔 지수 ETF 적립식처럼 단순한 구조부터
- [ ] ISA 안에서 손익통산 — 수익·손실이 합산돼 세금 계산

## STEP 4 · 유지 & 절세

- [ ] 3년 의무 유지 — 중도 해지하면 세제 혜택 반납
- [ ] 급하면 원금 내 중도인출 가능 (혜택 유지) — 해지 대신 인출
- [ ] 만기 후 연금계좌 이전 시 추가 세액공제 — 은퇴 준비와 연결
